------------------------- RECONOCIMIENTO DE VOZ: INSTALACIÓN -----------------------------------

1) Instalar speech recognition

	sudo pip3 install SpeechRecognition

2) Instalar dependencias

	sudo apt-get install python-pyaudio python3-pyaudio

3) Si no va la instalación de las dependencias anteriores, o siguen faltando dependencias.

	sudo apt-get install libasound-dev
	
	Descargar portaudio desde: http://files.portaudio.com/download.html
	
	Instalarlo haciendo uso del makefile (make install)
	
4) Si sigue dando errores

	sudo apt-get install libasound2
	
	sudo apt-get install libasound2:i386
	
Con estos pasos a mí ya me funcionaba, aunque al usar el reconocimiento siguen saliendo mensajes de error pero aun así funciona.
